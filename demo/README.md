# Claude Hub Public Demos

This directory contains publicly accessible demos and visualizations generated by Main Claude.

## Setup

Served via nginx with HTTPS (Let's Encrypt SSL):
- **URL:** https://claude-hub.abstractionlair.xyz/demo/
- **Config:** `/etc/nginx/sites-available/claude-hub` (location /demo block)
- **Permissions:** Directory is 755, parent `/home/claude` has o+x for nginx traversal

## Adding New Demos

To make output publicly accessible:

1. Create your HTML/visualization file
2. Copy it to `/home/claude/claude-hub/demo/`
3. Ensure file permissions are 644: `chmod 644 your-file.html`
4. Update `index.html` to include the new demo
5. Access at `https://claude-hub.abstractionlair.xyz/demo/your-file.html`

No server restart needed - nginx serves files directly.

## Usage Pattern

This is the standard location for:
- Research reports (like Polymarket analysis)
- Interactive visualizations
- Data dashboards
- Any output that should be web-accessible

Do NOT create ad-hoc HTTP servers on random ports. Use this directory instead.
