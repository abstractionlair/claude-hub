# Claude Hub Public Demos

This directory contains publicly accessible demos and visualizations
generated by the hub's main Claude session.

## Setup

Serve this directory as static files behind your web server (any static
`location` block works; no server restart needed for new files).

## Adding New Demos

1. Create your HTML/visualization file
2. Copy it into `demo/`
3. Update `index.html` to include the new demo

## Current Demos

- **index.html** - Landing page listing all demos
- **polymarket-analysis.html** - Backtesting analysis (Jan 2026)
- **neural-stream.html** - Interactive thought visualization (Dec 2025)

## Usage Pattern

This is the standard location for research reports, interactive
visualizations, data dashboards — any output that should be
web-accessible. Do NOT create ad-hoc HTTP servers on random ports; use
this directory instead.
